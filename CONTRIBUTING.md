# Contributing to AI-Native Review

Thanks for your interest! 🎉

## Ways to Contribute

- **New anti-patterns** — Found a pattern that should be flagged? Open an issue or PR
- **Domain rules** — Add domain-specific exemptions (healthcare, legal, etc.)
- **Examples** — Share real review reports (anonymized if needed)
- **Bug fixes** — For the review logic or report generation

## Getting Started

1. Fork the repo
2. Clone: `git clone https://github.com/{your-username}/ai-native-review.git`
3. Create a branch: `git checkout -b feature/your-idea`
4. Make changes
5. Test: Run the skill against a real tool design
6. Submit a PR

## Good First Issues

Look for [`good first issue`](https://github.com/gtskevin/ai-native-review/labels/good%20first%20issue) labels.
